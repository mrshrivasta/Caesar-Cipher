What is Caesar Cipher?

A Caesar Cipher is a classic substitution cipher where each letter is moved by a fixed number of places in the alphabet. It is simple, memorable, and great for learning how encryption works.

How it works

Choose a shift from 1 to 25. Encryption moves letters forward by that amount, while decryption moves them backward. Case is preserved, and non-letter characters are left untouched.

Limitations

Caesar Cipher is not secure for real secrets because there are only 25 possible shifts. The brute-force table can reveal every possible decoding almost instantly.
