## 🔎 About This Project

This project is a **browser-based cryptography tool** designed to demonstrate how the Caesar Cipher works. It is ideal for students, beginners in cybersecurity, and developers exploring encryption techniques.

---

## 🚀 Features

* Caesar Cipher encryption tool
* Caesar Cipher decryption tool
* Adjustable shift key
* Real-time text processing
* Lightweight and fast UI
* Fully client-side (no data sent to server)

---

## 🧠 What is Caesar Cipher?

The **Caesar Cipher** is a substitution cipher where each letter in the plaintext is shifted by a fixed number of positions in the alphabet.

**Example:**

HELLO → KHOOR (Shift = 3)
